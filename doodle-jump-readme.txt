# Schwarz-Weiß Doodle Jump

Ein einfacher Doodle Jump Klon in Schwarz-Weiß, geschrieben in HTML5 und JavaScript.

## Beschreibung

Schwarz-Weiß Doodle Jump ist eine minimalistische Version des beliebten Smartphone-Spiels Doodle Jump. In diesem Spiel steuerst du einen kleinen Charakter, der von Plattform zu Plattform springt und versucht, so hoch wie möglich zu kommen.

## Spielanleitung

1. Öffne `index.html` in einem Webbrowser
2. Verwende die Pfeiltasten links/rechts oder A/D-Tasten, um den Spieler zu steuern
3. Springe auf Plattformen, um nach oben zu gelangen
4. Vermeide, nach unten zu fallen
5. Erreiche einen möglichst hohen Punktestand!

## Funktionen

- Minimalistische Schwarz-Weiß-Grafik
- Einfache Steuerung
- Unbegrenztes Gameplay
- Automatischer Neustart nach Spielende
- Durchgängige Spielwelt (das Verlassen des Bildschirms auf einer Seite lässt dich auf der anderen Seite wieder erscheinen)

## Technische Details

Das Spiel wurde vollständig mit HTML5, CSS und JavaScript erstellt. Es verwendet das HTML5 Canvas-Element für die Grafik und JavaScript für die Spiellogik.

## Installation

1. Lade die `index.html`-Datei herunter
2. Öffne sie in einem beliebigen modernen Webbrowser

Alternativ kannst du das Repository klonen:

```
git clone https://github.com/deinbenutzername/schwarz-weiss-doodle-jump.git
cd schwarz-weiss-doodle-jump
```

## Anpassung

Du kannst das Spiel leicht anpassen, indem du die verschiedenen Konstanten im JavaScript-Code änderst:
- `gravity`: Ändert die Stärke der Schwerkraft
- `playerJumpStrength`: Ändert die Sprunghöhe des Spielers
- `playerSpeed`: Ändert die horizontale Bewegungsgeschwindigkeit
- `platformCount`: Ändert die Anzahl der Plattformen
